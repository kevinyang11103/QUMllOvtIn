# Python在线考试系统 python1273

## 项目概述

本项目是一款基于Python3的在线考试系统，采用前后端分离的架构设计，提供了学院班级管理、考试科目管理、学生信息管理、教师管理以及校园考试安排等完整功能。

## 技术栈

- 后端：Python3 + Django
- 前端：Vue
- 数据库：MySQL
- 开发工具：Pycharm, Navicat

## 运行环境

- Python版本：3.0及以上
- MySQL版本：8.0

## 功能模块

- 学院班级管理
  - 班级信息维护
  - 学生归属管理
- 考试科目管理
  - 科目信息设置
  - 考试时间配置
- 学生信息管理
  - 个人信息维护
  - 考试成绩查询
- 教师管理
  - 教师信息维护
  - 教师权限分配
- 校园考试安排
  - 考试时间安排
  - 考试场次管理

## 系统角色

- 学生
- 教师
- 管理员

## 目录结构

```plaintext
|-- 项目名称
    |-- backend    # 后端代码目录
    |-- frontend   # 前端代码目录
    |-- db         # 数据库相关文件
    |-- scripts    # 脚本文件目录
    |-- ...
```

## 部署步骤

1. 环境准备：安装Python3.0以上版本和MySQL8.0数据库。
2. 数据库配置：根据`db`目录下的文件建立数据库及表格。
3. 后端部署：进入`backend`目录，按照项目说明进行配置和运行。
4. 前端部署：进入`frontend`目录，构建前端资源，并按照项目文档部署到Web服务器。

## 核心亮点

- 完善的考试流程管理功能，覆盖考试全周期。
- 前后端分离，支持多端适配，易于维护和扩展。
- 灵活的权限控制，保障系统安全与数据隔离。
